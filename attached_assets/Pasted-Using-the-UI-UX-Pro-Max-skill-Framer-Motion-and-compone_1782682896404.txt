Using the UI UX Pro Max skill, Framer Motion, and components inspired by 21st.dev, completely upgrade the visual design of MyoMap. This is a full UI overhaul.

## Setup
- Install Framer Motion: npm install framer-motion
- Install Lucide React if not already installed: npm install lucide-react
- Use Google Font "Plus Jakarta Sans" — add to index.html:
<link href="https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@400;500;600;700;800&display=swap" rel="stylesheet">
- Apply globally: font-family: 'Plus Jakarta Sans', sans-serif

## Brand colors
- Primary teal: #0D9488
- Dark slate: #1E293B
- Background: #0a0f1a
- Card background: #111827
- Border: rgba(13, 148, 136, 0.15)
- Remove ALL blue from the app — every accent must be teal
