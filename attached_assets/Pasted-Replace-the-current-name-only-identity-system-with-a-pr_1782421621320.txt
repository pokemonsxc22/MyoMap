Replace the current name-only identity system with a proper email + password authentication system. Here's exactly what to build:

## Auth flow
- Use Supabase's built-in email + password auth (signUp, signInWithPassword, signOut, resetPasswordForEmail)
- On sign up: collect first name, email, and password
- On sign in: email and password
- On forgot password: send a reset link to their email via Supabase

## Supabase users table
Make sure this SQL has been run (skip if table already exists):
CREATE TABLE IF NOT EXISTS users (
  id UUID PRIMARY KEY,
  name TEXT NOT NULL,
  email TEXT,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

On sign up, insert a row into users table using the Supabase auth user's UUID as the id, with their name and email.
