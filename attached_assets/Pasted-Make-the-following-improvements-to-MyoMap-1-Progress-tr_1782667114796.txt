Make the following improvements to MyoMap:

## 1. Progress tracking — session feedback after completing a routine
On the dashboard under "Track Your Progress", add a session feedback card that users can fill out after completing their exercises. It should include:

- A difficulty rating: "How hard were the exercises?" with 3 options displayed as selectable buttons: Easy / Moderate / Hard
- A progress rating: "Did your pain/mobility improve?" with 3 options: Better / Same / Worse
- A short text area: "Anything else to note?" (optional, max 300 characters)
- A "Save Progress" button (teal, full width)
- Save this feedback to Supabase in a new table called `progress_logs` with columns: id (uuid), user_id (uuid), assessment_id (uuid), difficulty (text), improvement (text), notes (text), created_at (timestamp)
- Show a history of past progress logs below the feedback form, grouped by date, showing difficulty, improvement, and notes for each session
- Run this SQL in Supabase to create the table:

CREATE TABLE IF NOT EXISTS progress_logs (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL,
  assessment_id UUID,
  difficulty TEXT,
  improvement TEXT,
  notes TEXT,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
ALTER TABLE progress_logs ENABLE ROW LEVEL SECURITY;
CREATE POLICY "progress_logs_user_access" ON progress_logs
  FOR ALL USING (auth.uid() = user_id) WITH CHECK (auth.uid() = user_id);
GRANT ALL ON progress_logs TO authenticated;
